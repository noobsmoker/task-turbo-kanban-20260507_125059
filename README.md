# Task-Turbo Kanban

High-performance task management and productivity tool - kanban edition.

## Features
- Lightweight and fast performance
- Cross-platform compatibility  
- Simple command-line interface
- Extensible architecture

## Installation
```bash
git clone https://github.com/noobsmoker/task-turbo-kanban-20260507_125059.git
cd task-turbo-kanban-20260507_125059
chmod +x taskturbo.py
```

## Usage
```bash
python taskturbo.py --help
```

## Requirements
- Python 3.6+
- Standard library only
